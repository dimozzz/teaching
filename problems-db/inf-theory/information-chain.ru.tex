(Chain rule). Докажите, что имеют место следующие соотношения:
\begin{enumcyr}
    \item $I((\alpha, \beta) : \gamma) = I(\alpha : \gamma) + I(\beta: \gamma \mid \alpha)$.
    \item $I((\alpha, \beta) : \gamma \mid \delta) = I(\alpha : \gamma \mid \delta) + I(\beta: \gamma \mid
        \alpha, \delta)$.
\end{enumcyr}
