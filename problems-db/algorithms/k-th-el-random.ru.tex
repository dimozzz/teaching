Придумайте вероятностный алгоритм, находящий $i$-й по величине элемент данного массива длины $n$,
используя случайную выборку элементов массива. Время работы должно быть линейным с <<хорошей>>
константой.

\hinttext{Комментарий:} с плохой константой можно и детерминированно.

\answer{
    Random sampling. Определим верхнюю и нижнюю границы, посчитаем количество элементов, меньших нижней
    границы.
}
