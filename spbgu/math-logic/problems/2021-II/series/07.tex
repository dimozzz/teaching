\begin{definition*}
    Пусть $A$~--- $\sigma$-структура, $X \subseteq A$, и $\sigma_x \coloneqq (\sigma, \{b^x \mid x \in
    X\})$~--- расширение сигнатуры $\sigma$ при помощи новых констант для каждого элемента множества
    $X$. $A_X$, соответственно, расширение $A$, где элементы $b^x$ интерпретируются естественным образом
    (т.е для любого $a \in X$ константа $b^a$ интерпретируется, как $a$).

    \deftext{Частичным $n$-типом} (или просто $n$-типом) $\sigma$-структуры $A$ над множеством $X$
    называть типы теории $\Theory(A_x)$, соответствющим образом перенесём понятия \deftext{полного типа}
    и \deftext{реализации типа}. По умолчанию будем считать множество $X$ пустым.
\end{definition*}

}
