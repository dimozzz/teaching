\task{
    Распознается ли язык $L \coloneqq \{a^{3n} b^{3m} \mid n, m \in \mathbb{N}\}$ детерминированным
    конечным автоматом?
    \answer{да}
}
%да

\task{
    Является ли язык правильных скобочных последовательностей регулярным?
    \answer{нет}
}
%нет

\task{
    Является ли множество машин Тьюринга, останавливающихся на как минимум двух входах, перечислимым?
    \answer{да}
}
% да

\task{
    Верно ли, что любое подмножество разрешимого множества является разрешимым?
    \answer{нет}
}
% нет

\task{
    Является ли множество машин Тьюринга, останавливающихся на всех четных входах, перечислимым?
    \answer{нет}
}
% нет

\task{
    Верно ли, что $\RP \subseteq \PSPACE$?
    \answer{да}
}
%да

\task{
    Пусть $\NP \neq \coNP$. Верно ли, что $\P \neq \NP$?
    \answer{да}
}
%да

\task{
    Существует ли такая пара распределений $\alpha, \beta$, что $\entropy(a) < \entropy(a \mid \beta)$?
    \answer{нет}
}
%нет

\task{

    \answer{2}
}
